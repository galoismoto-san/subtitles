# The AI Agent Economy Is Here

**YouTube:** https://youtu.be/Q8wVMdwhlh4  
**Podcast:** The Light Cone — Gary Tan & Harj Taggar

---

## 摘要

這集討論 AI Agent 經濟的到來。

### 1. Claude Code / OpenClaw 接管開發者生活
非技術 CEO 開始全面自動化業務流程；工程師背景的 CEO 重拾寫程式，一次跑 4–5 個 agent 並行工作到凌晨 2–3 點。

### 2. Moltbook — 第一個 AI Agent-Only 社群
YC 校友打造，agents 在無人類參與下自行互動。兩天內的內容量即超越 Reddit 前兩年，因為 LLM 可以超人類速度產出文字。

### 3. Agent 經濟的形成
Agents 會自主選擇 dev tools，顛覆傳統 go-to-market 邏輯：
- 過去：開發者透過 Stack Overflow、GitHub trending、口耳相傳選工具
- 現在：agents 讀文件、選 API，文件品質 = 被選中的關鍵
- 案例：Supabase 因 agents 預設選其 Postgres 文件而爆炸性成長

### 4. YC 格言升級
> "Make something people want" → **"Make something agents want"**

### 5. 開發者市場大爆炸
從 ~2000 萬訓練工程師，擴展至全球數億 vibe coding 使用者，加上 agents 本身作為獨立經濟行為者。

### 6. 給 Founders 的建議
- 親手使用 agents，建立直覺（"cyber psychosis" 是必要的）
- 理解模型的自然傾向，順著做而非對抗
- 優先提供開放 API — agents 討厭網站 UI
- 以模型視角設計工具：讓 agent 想用你的產品
